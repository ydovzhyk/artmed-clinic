# ARTMED Clinic Website

Modern landing page for a medical clinic in Spain (Platja d’Aro), focused on Russian-speaking clients and local Spanish search visibility.

## 🌍 Overview

ARTMED is a medical clinic offering a wide range of services including consultations, aesthetic medicine, rehabilitation, and modern diagnostic procedures.

This project represents a production-ready landing page with strong focus on:
- clean UI/UX
- SEO optimization
- structured data (schema.org)
- responsive design

## 🚀 Tech Stack

- Next.js (App Router)
- React
- Tailwind CSS (v4)
- Custom UI components
- Schema.org structured data (MedicalClinic)

## ✨ Features

- 📱 Fully responsive design
- 🌐 Multilingual SEO targeting
- 🔍 Optimized metadata and OpenGraph
- 🧠 Structured data (JSON-LD for MedicalClinic)
- ⚡ Fast performance and optimized assets
- 🎨 Custom design system with Tailwind

## 📍 Location

Platja d’Aro, Spain
Target audience: Russian-speaking patients and local Spanish users
